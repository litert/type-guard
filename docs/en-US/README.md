# TypeGuard Documentation Index (en-US)

This documentation set is organized around practical TypeGuard usage scenarios, following the path of “Quick Start → API Reference → Syntax Details”.

## Suggested Reading Order

1. [Quick Start](./quick-start.md)
2. [API Reference](./api-reference.md)
3. [Syntax Guide](./syntax-guide/README.md)

## About These Docs

- **Quick Start**: The shortest path to install, compile, and run.
- **API Reference**: Detailed usage of creating and using `Compiler` / `InlineCompiler`.
- **Syntax Guide**: Complete syntax rules (built-in types, structural types, modifiers, string matching, syntax sugar, filters, and more).

## Version Scope

These docs are based on the implementation and test cases under the repository’s `src/` directory, covering features released up to `v1.4.2`.
